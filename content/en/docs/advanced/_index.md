# Advanced Features

Podstack offers advanced capabilities for power users, including launch templates, serverless GPU notebooks, and AI Studio for model management.

## In This Section

- **[Templates](/docs/advanced/templates/)** - Save and reuse pod configurations
- **[Serverless GPU](/docs/advanced/serverless-gpu/)** - Quick Jupyter notebooks with GPU
- **[AI Studio](/docs/advanced/ai-studio/)** - Model fine-tuning, evaluation, and deployment

## Feature Overview

### Launch Templates

Save pod configurations for quick deployment:
- Store resource specifications
- Save environment variables and ports
- Deploy identical pods with one click
- Share configurations within projects

[Learn more](/docs/advanced/templates/)

### Serverless GPU

JupyterHub-based notebooks with GPU access:
- Quick startup without configuration
- Pre-installed ML libraries
- Automatic resource management
- Pay only for active time

[Learn more](/docs/advanced/serverless-gpu/)

### AI Studio

End-to-end ML model management:
- Browse model catalogue
- Fine-tune base models
- Evaluate model performance
- Deploy models for inference

[Learn more](/docs/advanced/ai-studio/)

## Feature Availability

Some advanced features may require:
- Specific account types
- Feature flags enabled
- Minimum wallet balance

Contact support if features aren't visible in your portal.
